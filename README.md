# DG_Sample_Code_Check

Healthcare Benefits Correspondence Automation System — Pick/MultiValue (UniData) platform analysis.

## System Analysis Report

- **HTML Report (GitHub Pages):** [https://sachetcognition.github.io/DG_Sample_Code_Check/](https://sachetcognition.github.io/DG_Sample_Code_Check/)
- **Word Document:** [docs/DG_Analysis_Report.docx](docs/DG_Analysis_Report.docx)

The report covers:
- Record type analysis (10 record types with key formats, field tables, access patterns)
- 9 letter types identified from code (COBRA, HIPAA, Disabled Dependent)
- Dependency graph and coupling analysis
- Decoupling strategy with 8 proposed extracted subroutines
- Program flow diagrams for COBRAN, HIPAA.NPP, and PRINT.DISABLED.DEP.LETTERS
